Exercises (Specialized & Workplace; Redundancy & Backup)

Task 1 (4 points). Mind Map v2

Use FreeMind mind map drawing tool (available in our Moodle) to draw a mind map of the Lecture 11. Each topic should be a separate branch, and its subtopic become sub-branches. Save the resulting mind map as a FreeMind file lecture11.mm and as a PNG picture lecture11.png.

Task 2 (2 points). You are the Commander v2

Get muCommander file manager from our course Moodle. Its functionality is limited, but it works fine on all our platforms. Use muCommander to:

  1. Unpack the FreeMind tool archive.
  2. Go inside accessories subfolder inside FreeMind folder.
  3. Select all xsl files and copy them into some empty folder.
  4. Rename them all into 01.xsl, 02.xsl, … (use File, Batch rename operation).
  5. Pack them into a Tar/Gzip archive archive.tar.gz.

Task 3 (4 points). RAID Types

Read a Wikipedia article on standard RAID levels. If you decide to purchase a RAID array for your personal home use, which RAID level would you choose? Why? Summarize your reasons in a short report.

Submission Instructions.

Place the files lecture<NN>.mm, lecture<NN>.png (task 1), archive.tar.gz (task 2) and a text report file (task 3) into a single archive <your-student-ID>-ex3.zip. Upload the archive to the course Moodle system.
